# ПРОЙДУ? — национальный каталог вузов

Telegram Mini App для поиска вузов России и проверки поступления по подтвержденным данным.

## Текущий релиз

- интро полностью удалено;
- каталог подключен к 89 региональным страницам официального мониторинга высшего образования;
- поиск по названию, городу и филиалам;
- общий режим «Вся Россия» с локальным кэшем;
- поиск по сумме баллов и по направлению сохранен;
- конкурсные расчеты выполняются только по проверенным программам (текущий пилот — КБГУ);
- challenge-ссылки работают через `@proidu2026_bot`.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm test
npm run build
```

## GitHub Pages

Production build создается командой `npm run build` с относительным base path. Содержимое `dist/` загружается в корень ветки GitHub Pages.
